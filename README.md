# Jable Downloader

## 使用工具

1. App 框架: Electron
2. UI 框架及衍生生態系: React / RTK / Styled-component / Chakra-UI
3. 爬蟲: cheerio
4. 打包工具: Webpack

## APP FEATURE

1. 即時網頁內容爬蟲
2. 下載 / 解碼 mp4 檔案
3. 新增深色模式
4. 新增安全模式
